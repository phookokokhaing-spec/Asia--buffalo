Lobby HTML အစီအစဉ်ပြန်စီရန်

1. reorder_lobby.py ကို lobby.html ရှိတဲ့ folder ထဲထားပါ။
2. Terminal မှာ run ပါ:

   python reorder_lobby.py lobby.html lobby_ordered.html

3. lobby_ordered.html ကိုဖွင့်စစ်ပါ။

ဒီ script က id, onclick, class, image path နဲ့ content တွေကိုမပြောင်းဘဲ top-level section နေရာပဲ ပြန်စီပါတယ်။ Script imports တွေကို အောက်ဆုံးမှာ မူလ load order အတိုင်းထားပါတယ်။
